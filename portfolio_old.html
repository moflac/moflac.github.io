RailTrack

 

RailTrack was an Android app showing real time passenger train locations of Finnish railways, VR on map. It also displayed train and station specific timetables and traffic bulletins. It used "Junat kartalla" API provided by VR and Google Maps SDK for Android. Discontinued after API was changed and transferred to Traffic Management Finland. Source code (in Java): RailTrack marko-ohra-aho


360° stereo web viewer

An experiment to make a web based stereo panorama viewer for pictures captured with Google's Cardboard Camera app.
Uses A-Frame, which is a JavaScript web framework for building VR environments. Stereo viewing works with mobile web browsers like Chrome and Firefox.
Monoscopic viewing also with desktop browsers. Has been tested and worked with Oculus Rift.
