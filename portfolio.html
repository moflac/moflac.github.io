GPS and accelerometer logger

A small Android app to show your device's current location coordinates using Fused Location Provider API.
Shows also current linear acceleration values.
Made with Android Studio, Android SDK, Java.
Can record location and acceleration values to a csv file on your device. Works on Android version 8.0 and higher.
Source code in GitHub: LocAcc


RailTrack

RailTrack was an Android app showing real time passenger train locations of Finnish railways, VR on map. It also displayed train and station specific timetables and traffic bulletins. It used "Junat kartalla" API provided by VR and Google Maps SDK for Android. Discontinued after API was changed and transferred to Traffic Management Finland. Source code (in Java): RailTrack

Raspberry Pi weather station

A very light weight temperature logger. A Python script records room temperature from a temperature sensor and outside temperature from a weather API.
Results are saved to a constant size round robin database. Rrdtool is used to generate a graph in png format which can be used in a web page. Lighttpd is used as the web server. Source

  • Lighttpd
  • Python, PHP, html
  • RRDTool


HandsOn

Research project at Tampere University of Technology. A part of the project was to create and test a haptic glove utilizing pneumatic artificial muscles. The device used one PAM for each finger in one hand and was controlled with a PC. The control PC was connected to a computer cluster driving a VR environment.

  • VR environment created and running with Virtools
  • haptic glove control and interface to Virtools with C++ and VRPN
  • PC cluster running NVidia Quadro cards


Simulator sickness reduction

Research project at Tampere University of Technology aimed at reducing simulator sickness caused by a bus simulator to its users. Experiments aimed to reduce simulator sicness were conducted after thorough literature review on subject. Successful means to reduce sickness were found and implemented in bus simulator training. Master's thesis made from the research: Simulaattorisairauteen vaikuttavien tekijöiden vähentäminen bussisimulaattoriympäristössä
